# ludob99.github.io

Personal portfolio site, live at **[lbelzile.ca](https://lbelzile.ca)**.

## Stack

Plain HTML, CSS, and vanilla JavaScript. No build step, no dependencies.

Bilingual (English / French) via a lightweight client-side i18n system driven by `data-i18n` attributes and a `?lang=` query parameter.

## Run locally

Open `index.html` directly in a browser.
